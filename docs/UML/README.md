# 📐 DrawBoard UML 设计图

本目录包含 DrawBoard 系统的 UML 设计图，基于 Mermaid 语法。

## 📋 设计图清单

| 设计图 | 说明 |
|--------|------|
| [DESIGN.md](./DESIGN.md) | 完整的 UML 设计图集合 |

## 🎯 包含内容

1. **整体架构图** - 六层架构分层（新增基础设施层）
2. **核心类图** - 类结构与关系（包含 EventBus、SelectToolCoordinator）
3. **绘制时序图** - 绘制流程（含 EventBus 事件发布）
4. **选择工具时序图** - 含脏矩形优化的拖拽流程 ⭐
5. **EventBus 事件流图** - 组件间通信 ⭐
6. **模块依赖图** - 模块间依赖关系
7. **图层结构图** - 物理层与虚拟层映射
8. **SelectTool 子模块架构图** - 11 个子模块关系 ⭐
9. **设计模式应用图** - 6 种设计模式
10. **缓存层级图** - 5 级缓存架构 ⭐

## 🆕 v4.0 新增内容

- **EventBus 事件总线** - 类型安全的组件间通信
- **SelectToolCoordinator** - 选择工具协调器
- **脏矩形优化** - 局部重绘时序
- **缓存层级** - 5 级缓存架构可视化
