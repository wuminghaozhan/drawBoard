# 🎨 DrawBoard - 专业级绘图板系统

## 📋 项目概述

DrawBoard 是一个基于 Canvas 的高性能绘图解决方案，提供完整的绘制工具链和现代化的用户体验。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

访问 http://localhost:3000 查看应用。

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript

- **样式**: SCSS + CSS Modules

- **构建工具**: Create React App
- **包管理**: npm

## 🔧 开发指南

### 添加新功能页面

1. 在 `src/pages/` 下创建新的页面组件
2. 在 `src/App.tsx` 中添加路由配置
3. 在 `src/components/Navigation.tsx` 中添加导航链接

### 样式开发

- 使用 SCSS 进行样式开发
- 遵循 BEM 命名规范
- 支持响应式设计
- 使用 CSS 变量进行主题管理

### 组件开发

- 使用 TypeScript 进行类型安全开发
- 遵循 React Hooks 最佳实践
- 组件职责单一，高度可复用
